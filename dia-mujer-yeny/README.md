# Tarjeta Día de la Mujer · Yeny

Mini sitio web estático listo para GitHub Pages.

## Incluye
- Animación inicial 3D con flores
- Mensaje para Yeny con estilo de amistad y admiración
- Variantes aleatorias cada vez que se abre
- Botón para descargar la tarjeta en PNG
- Compatible con GitHub Pages

## Publicación rápida
1. Sube estos archivos a un repositorio.
2. Activa GitHub Pages en **Settings > Pages**.
3. Selecciona la rama principal y la carpeta raíz.
4. Guarda.

## Archivos
- `index.html`
- `styles.css`
- `script.js`
